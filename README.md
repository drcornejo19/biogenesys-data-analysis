# Biogenesys – Health Data Analysis Project

## 📌 Project Overview
This project analyzes historical health data from Latin American countries to identify patterns and support strategic expansion decisions.

## 🎯 Objective
Perform exploratory data analysis (EDA) and data cleaning to uncover insights related to mortality rates, health indicators and country-level performance metrics.

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 🔎 Key Tasks
- Data cleaning and preprocessing
- Handling missing values
- Exploratory Data Analysis (EDA)
- Data visualization
- Country comparison analysis

## 📊 Key Insights
- Identification of health performance differences between countries
- Trend detection across years
- Data inconsistencies and outlier detection

## 🚀 Outcome
Structured analytical insights to support data-driven healthcare expansion strategies.

## 📊 Power BI Dashboard

This project also includes an interactive Power BI dashboard designed to translate analytical findings into strategic decision support.

### Dashboard Focus
- Comparative country-level performance
- Mortality trends and health indicators
- Strategic expansion opportunity mapping
- Visual KPI monitoring
